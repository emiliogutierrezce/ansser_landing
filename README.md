# Landing Page - Ansser

Esta es la landing page oficial para Ansser, un software con chatbot integrado para automatizar ventas por mensajes usando inteligencia artificial.

## Descripción del Proyecto

El sitio web está construido con HTML y CSS puro, con una estructura modular para facilitar el mantenimiento y las futuras actualizaciones. El diseño fue creado en Figma y el código fue optimizado para ser semántico, accesible y responsivo.

### Estructura

-   **`index.html`**: Contiene toda la estructura semántica de la página.
-   **`assets/css/`**: Contiene los estilos, organizados en una arquitectura modular (base, componentes, secciones).
-   **`assets/images/`**: Contiene todos los recursos visuales (logos, íconos, imágenes).

### Cómo visualizar

Simplemente abre el archivo `index.html` en cualquier navegador web.
